Configuration des equipes

Liste des equipes competition synchronisees avec la FFF. A mettre a jour en debut de saison (aout) quand la FFF attribue les nouveaux identifiants. Les libelles d'affichage peuvent etre modifies a tout moment.

Equipes

L'enregistrement vide le cache FFF des classements et resultats. Les donnees seront re-telechargees a la prochaine visite.

Passage a la nouvelle saison

A faire chaque annee en aout/septembre, quand la FFF publie les nouveaux identifiants. Cette action :

  • Incremente la saison (ex: 2025-2026 → 2026-2027)
  • Pre-remplit les nouveaux IDs FFF en remplacant le millesime (ex: 2025_8748_SEM_1 → 2026_8748_SEM_1)
  • Archive l'ancienne config dans data/equipes-archive/

Tu devras ensuite verifier les IDs sur le site FFF (le numero d'equipe peut changer si l'effectif evolue) et appuyer sur "Enregistrer la configuration".
